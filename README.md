# 🧠 Next Word Prediction App

A simple and clean **Streamlit web app** that predicts the next word using an **LSTM model**.

---

## 🚀 Features

- 🔮 Next word prediction
- 🤖 LSTM deep learning model
- ⚡ Fast & interactive UI
- 🖤 Clean black & white design

---

## 🛠️ Tech Stack

- Python
- Streamlit
- TensorFlow / Keras
- NumPy

---

## ▶️ Run the App

```bash
pip install streamlit tensorflow numpy
streamlit run app.py
